Personal Projects


Spotify Unlocked (code)
Spotify Unlocked is a Ruby on Rails web application that integrates with the Spotify API. It can:
  • Get detailed information about any song, artist or album on Spotify.
  • See Spotify user data, including hidden gems, highest energy favorite songs, and more.
  • Construct a chord representation of a song
  • Get song recommendations based off specific qualities

Tolkein Text (code)
Tolkein Text implements a Neural Network Language Model in Pytorch. It can:
  • Train an LSTM Neural Network Language Model on The Lord of the Rings.
  • Use the trained model to generate new text based on LOTR
  • Preprocess and tokenize raw text for use as training data
  • For more details on implementation check out this README
