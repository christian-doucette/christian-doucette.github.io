Personal Projects


Spotify Unlocked (code)
Spotify Unlocked is a Ruby on Rails web application that integrates with the Spotify API. It can:
  • Get detailed information about any song, artist or album on Spotify.
  • See Spotify user data, including hidden gems, highest energy favorite songs, and more.
  • Construct a chord representation of a song
  • Get song recommendations based off specific qualities (in progess)

Tic-Tac-Toe Bot (code)

This Bot plays tic-tac-toe using the Minimax algorithm to calculate the optimal next move.
Since an optimal player can always at least force a draw, it will never lose. Built with Javascript.


QuoteTabs (code)

QuoteTabs is a Chrome extension that displays a math or computer science quote on new tabs.
It gets the quotes from one of three different random quote APIs.
